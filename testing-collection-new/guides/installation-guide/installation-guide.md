# Installation Guide

Complete guide for installing Testing Collection across different platforms and environments.

## Table of Contents

* [System Requirements](./system-requirements.md)
* [Installation Methods](./installation-methods.md)
* [Troubleshooting](./troubleshooting.md)

## Quick Installation

### Using npm

```bash
npm install -g testing-collection
```

### Using yarn

```bash
yarn global add testing-collection
```

### Using pnpm

```bash
pnpm add -g testing-collection
```

## Platform-Specific Instructions

### macOS

```bash
# Using Homebrew
brew install testing-collection

# Or using npm
npm install -g testing-collection
```

### Linux

```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install testing-collection

# Or using npm
npm install -g testing-collection
```

### Windows

```powershell
# Using npm
npm install -g testing-collection

# Or using Chocolatey
choco install testing-collection
```

## Verification

Verify your installation:

```bash
testing-collection --version
```
